# Taurux App — نموذج التصميم الجديد (Interactive Prototype)

نموذج تفاعلي عالي الدقة لإعادة تصميم تطبيق Taurux + كل الميزات الجديدة المقترحة.
HTML/CSS/JS خالص — بدون أي تبعيات أو خطوة بناء.

## التشغيل / المعاينة

```bash
cd taurux-app-flutter/design_prototype
python -m http.server 8088        # أو:  npx serve -l 8088
```

ثم افتح: **http://localhost:8088/**

- اللوحة اليسرى = فهرس كل الشاشات (رئيسية / فرعية / ميزات جديدة).
- زرّا الثيم واللغة (عربي/EN، ليلي/نهاري) أعلى اللوحة.
- رابط مباشر لأي شاشة عبر الهاش: `…:8088/#rewards` ، `#progress` ، `#aiCoach` …

## محتوى النموذج (19 شاشة)

| المجموعة | الشاشات |
|---|---|
| البداية | الترحيب (Onboarding)، تسجيل الدخول (OTP + بصمة) |
| الرئيسية | الرئيسية، نشاطي، المتجر، حسابي |
| التفاعل | النقاط والمكافآت، الإنجازات والسلسلة، التحديات والمتصدرون، ادعُ صديقاً |
| اللياقة | تقدّمي (قياسات + صور)، تماريني (سجل + خطة)، المدرب الذكي AI |
| فرعية | حجز الحصة، ملف المدرب، الدفع، إدارة الاشتراك (تجميد/تجديد/ترقية)، بطاقة الدخول/المحفظة، الإشعارات |

> ملاحظة: البيانات هنا تجريبية (mock) لغرض استعراض التصميم فقط. الخطوة التالية تحويل
> الشاشات المعتمدة إلى Flutter وربطها بالـ backend الحقيقي.
